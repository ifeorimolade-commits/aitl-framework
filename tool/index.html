AITL · v0.1

Accessibility in the Loop (AITL) is a tool for evaluating the accessibility of AI-generated user interfaces. It applies the AI-Generated Interface Accessibility Assurance Framework (AIAAF): the six dimension rubric, the generative failure pattern taxonomy (P1 to P8), the 0 to 5 scoring scale, and the severity driven release gate. Run the automated scan first, then complete the human evaluation. Specification · Taxonomy · GitHub
