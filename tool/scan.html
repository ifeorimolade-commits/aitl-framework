AITL Scan · v0.1

Automated detection layer of Accessibility in the Loop (AITL), applying the AIAAF framework. Paste AI-generated markup; the engine detects the generative failure pattern taxonomy (P1 to P8) and structural defects automatically, fires the release gate, and routes what needs human judgment. Export the result into the AITL Evaluator to complete the human layer.

Input, AI-generated markup

New here? How to get a page's HTML code (30 seconds)
The easiest way to try this tool: click the Load sample button below. It fills in a real example so you can see a scan run immediately.

To check your own page:
  1. Open the page you want to check in your browser.
  2. Right-click anywhere on the page and choose View Page Source (or press Ctrl+U on Windows, Cmd+Option+U on Mac).
  3. Select all of it (Ctrl+A / Cmd+A), copy it, and paste it into the box below.
Working in an AI design tool? Copy the generated HTML straight from there and paste it in.
Have a live link instead of code? (advanced)

Browsers block most links for security, so this often will not work. If it does not, just paste the page's HTML in the box above instead (see the how-to). For scanning live links reliably, connect AITL to your AI assistant as an MCP server.
